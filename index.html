Command Line Access to Google Workspace

The Google Workspace CLI is a blazing-fast command-line interface written in Rust and distributed via npm. Announced in March 2026, it brings powerful automation capabilities to Gmail, Drive, Calendar, Sheets, Docs, and more — right from your terminal.

🦀 Built with Rust
📦 Distributed via npm
⚡ Lightning fast
🔒 Secure OAuth2

Quick Start

Install globally via npm:

npm i -g @googleworkspace/cli

After installation, authenticate with: gws auth login

Explore Commands by Service

Integration Patterns

🔗 Piping with Other Tools 

# Export Drive files to process with jq
gws drive list --format json | jq '.files[] | select(.size > 1000000)'

# Email yourself system metrics
df -h | gws gmail send --to me --subject "Disk Usage" --body-stdin

# Backup calendar to file
gws calendar list --days 30 > backup.json

🤖 OpenClaw/Claude Code Skills 

# Create a skill that emails reports
# In your Claude Code skill:

gws sheets read "Sales Report" A1:Z100 > report.csv
gws gmail send \
  --to team@company.com \
  --subject "Weekly Sales" \
  --body "See attached" \
  --attach report.csv

⚙️ Automation Scripts 

#!/bin/bash
# daily-backup.sh - Backup important docs

DATE=$(date +%Y-%m-%d)
gws drive download --folder "Important Docs" --dest ./backups/$DATE/
tar -czf backups-$DATE.tar.gz ./backups/$DATE/
gws drive upload backups-$DATE.tar.gz --parent "Backups"

🚀 CI/CD Integration 

# .github/workflows/report.yml
name: Daily Report
on:
  schedule:
    - cron: '0 9 * * *'

jobs:
  report:
    runs-on: ubuntu-latest
    steps:
      - run: npm i -g @googleworkspace/cli
      - run: gws gmail send --to team@co.com --subject Report
